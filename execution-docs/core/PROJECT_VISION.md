Current truth: the platform is intended to become a real, maintainable industrial e-commerce system, not a temporary storefront. It must support product management, orders, inventory, payment, admin operations and later integrations.

Implementation impact: every module should be designed as part of a growing commerce platform with clear ownership, traceable rules and room for future operational expansion.
